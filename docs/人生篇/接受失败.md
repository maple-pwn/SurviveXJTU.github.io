# 接受失败

> 笔者深夜失眠，阅读此书，发现竟然有一篇至今未更新的文章，遂起身，更之

在人生的前十八年里，我们几乎始终生活在一种明确的评价体系之中。分数、排名、结果，这些量化的指标不断告诉我们什么是“成功”，什么又意味着“失败”。在这种长期的训练之下，人们往往并不是在追求成长，而是在回避失败本身。

久而久之，我们开始在意数字的大小，开始担心一次失误是否会带来不可逆的后果，也开始习惯于用结果而非过程来审视自己。

在那段阶段里，失败似乎是不被允许的。

然而，当人真正进入大学，或者更广阔的人生阶段后，你会逐渐发现，这样的评价逻辑并不能完整地解释世界。现实生活中，很少存在绝对的成功，也很少存在真正意义上的失败。更多的时候，一次看似失败的经历，只是一次 **尚未完成的尝试**。

我们听过很多次“失败是成功之母”，但是否细想过为什么呢？

失败不再是一种否定，而是一种信息，一种提醒，一种对方向与方法的修正。

或许你会担心：一次失败，是否会影响未来的选择，甚至改变人生的轨迹。但如果回头看一看自己的经历，往往会发现，许多当时以为难以承受的时刻，最终都已被时间轻轻带过。所谓“轻舟已过万重山”，大多并非事先的笃定，而是事后的回望。

在大学这样一个相对宽容的环境中，个人的失败往往并没有想象中那样重要。它既不会被反复追究，也很少被长期记住。真正被保留下来的，反而是从失败中获得的经验与判断。

因此，与其花费大量精力去避免失败，不如尝试学会接受它。因为在更长的时间尺度下，个人的一次失败几乎不具备决定性意义；而在有限而宝贵的大学生活中，它却可能成为最真实、也最有效的积累。

倘若一切都能一蹴而就，人生未免过于平滑而舒适了些。来到这里的你，真的甘心如此吗？
